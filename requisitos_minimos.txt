REQUISITOS MINIMOS - APLICACAO GERENCIAMENTO

1) Linguagem e runtime
- Python 3.12+ (projeto validado localmente com Python 3.13.12)
- pip para instalacao das dependencias

2) Framework e bibliotecas Python (requirements.txt)
- Django==6.0.2
- django-tables2==2.8.0
- whitenoise==6.12.0
- waitress==3.0.2
- xlrd==2.0.1
- asgiref==3.11.1
- sqlparse==0.5.5
- tzdata==2025.3

3) Banco de dados
- SQLite 3 (backend configurado no projeto)

4) Sistema operacional
- Windows 10/11 64 bits, Linux 64 bits (ex.: Ubuntu 22.04+) ou macOS 12+

5) Hardware minimo (uso basico)
- CPU: 2 nucleos
- Memoria RAM: 4 GB
- Armazenamento livre: minimo de 2 GB

6) Navegador
- Google Chrome ou Microsoft Edge atualizados
  (necessario para recursos de upload de pasta via webkitdirectory)

7) Rede
- Acesso a rede local para abrir a aplicacao (porta configurada no servidor)
- Internet para carregar Tabulator via CDN (unpkg.com), conforme templates atuais
